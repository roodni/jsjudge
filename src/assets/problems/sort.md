次の表に示すプロパティを持つ、科目を表すオブジェクトを考えます。

| キー | データ型 | 説明 |
| --- | --- | --- |
| `name` | string | 科目名 |
| `score` | number | 点数 |

科目を表すオブジェクトの配列`subjects`が引数で与えられます。

`subjects`の科目を点数が低い順に並び替えて、科目名の配列を返してください。
ただし、点数が同じ科目が複数あった場合は、配列のインデックスが小さい方が前になるようにしてください。

関数の実行によって引数で与えた配列が破壊されても構いません。

```js
const subjects = [
  { name: "英語第一", score: 79 },
  { name: "英語第二", score: 65 },
  { name: "英語第三", score: 90 },
  { name: "英語第四", score: 60 },
  { name: "英語第五", score: 77 },
  { name: "英語第六", score: 77 },
  { name: "英語第七", score: 63 },
  { name: "英語第八", score: 80 }
];

f(subjects);  // ["英語第四", "英語第七", "英語第二", "英語第五", "英語第六", "英語第一", "英語第八", "英語第三"]
```
